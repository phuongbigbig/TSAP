Each row is a timepoint (enter its time in the first column). Each series has one or more replicate columns; the line shows the mean across replicates and the band shows its spread. Tip: copy a block of cells from Excel and paste onto any cell — values outside the table are ignored.

Accepted file format
The delimiter (comma or tab) is detected automatically. Two layouts are recognized:

1 · Wide (same shape as the table above) — first column Time; then one column per measurement. Group replicates of the same series by giving columns a shared name with an _R1, _R2… suffix (e.g. Time, Control_R1, Control_R2, Treated_R1, Treated_R2). Use _S1, _S2… instead to mark the same subjects tracked over time — this auto-enables repeated-measures mode.

2 · Long / tidy — columns named time, series (or group/condition), value, and optionally subject. One row per measurement; a subject column auto-enables repeated-measures.
